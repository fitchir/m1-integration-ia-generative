Questions a remplir :

- Ou se trouve la cle API ? (cote client ou cote serveur ?)
Reponse : La cle API se trouve cote serveur.

- Pourquoi est-ce une bonne pratique de garder la cle cote serveur ?
Reponse : C’est une bonne pratique car cela evite d’exposer la cle API au client ou a l’utilisateur. Cela protege l’acces au service et limite les risques d’abus ou d’utilisation non autorisee.
